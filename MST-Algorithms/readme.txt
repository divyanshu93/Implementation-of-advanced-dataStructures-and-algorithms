Prim1, Prim2 and Kruskals MST Algorithm

To run the Prim1 project
1. Open terminal
2. javac *.java
3. java Prim1
4. Enter the <path to the input file> on prompt, for ex: F:/LP2/src/sp0pq-big.txt

To run the Prim2 project
1. Open terminal
2. javac *.java
3. java Prim2
4. Enter the <path to the input file> on prompt, for ex: F:/LP2/src/sp0pq-big.txt

To run the Kruskals project
1. Open terminal
2. javac *.java
3. java Kruskals 
4. Enter the <path to the input file> on prompt, for ex: F:/LP2/src/sp0pq-big.txt

Output: 
Minimum Spanning Tree weight using the indicated algorithm.
